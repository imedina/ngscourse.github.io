Extract data in fastq format

Once you have downloaded one .sra file you can use the SRA Toolkit to extract the original reads and their qualities:

fastq-dump SRR926291.sra

But the SRA Toolkit also allows you reading the .sra file directly form the FTP site by simply using the file ID:

fastq-dump SRR926292

Extract data in SAM format

The SRA Toolkit allows you to get the SAM file of the run:

sam-dump SRR926291.sra > mysamfile.sam

And now you can use SAMtools to convert the file into a BAM file:

samtools view -Sb mysamfile.sam -o mysamfile.bam

You can perform the two previous steps at a time using the pipe operator |:

sam-dump SRR926291.sra | samtools view -Sb mysamfile.sam -o mysamfile.bam

You may want sam-dump to include SAM headers:

sam-dump --header SRR926291.sra > mysamfile_with_header.sam

Or to include your own text into the header:

sam-dump --header-comment 'THIS IS MY CUSTOM TEXT' SRR926291.sra > mysamfile_with_custom_header.sam

But watch out for the software the version:

sam-dump --version
